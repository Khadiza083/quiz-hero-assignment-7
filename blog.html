What is the difference between map, forEach and filter?

Ans:

Map returns an array. On the other hand, forEach returns undefined and filter checks every element of an array. If the condition is true then returns a new array.

What is the difference between localstorage, cookies and sectionStorage?

Ans:

Localstorage: It expires never. That means, if the browser is closed then the data will not be deleted. Capacity is 10MB

Cookies: It remains manually set and limited to 4KB. Storage locations are Browser and Server.

Sessionstorage: It expires when the tab is closed. Capacity is 5MB. Storage location is browser only
